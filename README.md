# MateoVenturas

Juego 2D en Unity inspirado en mi sobrino Mateo y su afición por los dinosaurios, creado como proyecto de clase para PMDM.

## Descripción
MateoVenturas es un proyecto de videojuego 2D desarrollado en Unity. El protagonista está inspirado en Mateo y la temática general del juego gira en torno a una aventura visual con elementos relacionados con dinosaurios.

## Tecnologías
- Unity 6
- C#
- Git
- GitHub

## Estructura principal

- ProjectSettings/
